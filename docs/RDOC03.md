```

REACT_APP_AUTH0_CLIENT_ID='HaDNmcczGrrwZ7SkPFTh5Owu3VDceltm'

REACT_APP_BACKEND_LINK='http://localhost:3001/signup'
```

Luego se debe ejecutar los comandos:

1. ```npm install```

2. ```npm start```
